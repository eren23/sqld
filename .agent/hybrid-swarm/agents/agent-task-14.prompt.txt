# Task: Volcano-model executor with all physical operators

Implement Executor trait: init/next/close/schema methods (pull-based Volcano model). Implement all physical operators: SeqScan (heap iteration with MVCC visibility, pushed-down filter), IndexScan (B+ tree walk, fetch heap tuples by TID, index-only scan), Filter (predicate evaluation), Project (expression projection), HashJoin (build/probe phases for INNER/LEFT/RIGHT/FULL/SEMI/ANTI, disk spill on work_mem overflow), SortMergeJoin (sorted merge), NestedLoopJoin (parameterized inner for index NL), Sort (in-memory quicksort, external merge sort with disk spill, multi-key ASC/DESC/NULLS FIRST|LAST), HashAggregate (hash table by GROUP BY, accumulators for COUNT/SUM/AVG/MIN/MAX/STRING_AGG/ARRAY_AGG/BOOL_AND/BOOL_OR with DISTINCT support), SortAggregate, Limit+Offset, Distinct (hash/sort), Union/Intersect/Except (ALL variants), Values (literal rows), Modify (INSERT/UPDATE/DELETE with index maintenance, constraint checking, WAL writes, RETURNING). Implement expression evaluator: compile to stack-based bytecode (ExprOp enum), evaluate per tuple. Implement all scalar functions: string (LENGTH/UPPER/LOWER/TRIM/SUBSTRING/POSITION/REPLACE/CONCAT/LEFT/RIGHT/REVERSE/LPAD/RPAD/REPEAT/SPLIT_PART), math (ABS/CEIL/FLOOR/ROUND/TRUNC/SQRT/POWER/MOD/LN/LOG/EXP/SIGN/RANDOM), date/time (NOW/EXTRACT/DATE_TRUNC/AGE/interval arithmetic/TO_CHAR), type (CAST/TYPEOF), null (COALESCE/NULLIF). Include comprehensive tests for every operator and function.
