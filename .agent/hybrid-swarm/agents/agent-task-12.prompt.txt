# Task: Semantic analyzer (binder) with name resolution and type checking

Implement the binder that transforms raw AST into bound/resolved AST. Phase 1 - Name resolution: resolve table names to catalog entries, column names to (table_id, column_ordinal), disambiguate unqualified columns (error if ambiguous). Phase 2 - Type checking: infer expression types bottom-up, apply coercion rules, validate operator type compatibility, check aggregate vs non-aggregate context in SELECT/HAVING. Phase 3 - Scope analysis: subquery scoping with outer column references for correlated subqueries, detect correlation depth. Phase 4 - Constraint checking: NOT NULL columns in INSERT, CHECK constraints on constants, FK target existence. Phase 5 - View expansion: replace view refs with stored SELECT AST, re-bind. Phase 6 - Wildcard expansion: SELECT * and t.* to explicit column lists. Output BoundSelect/BoundExpr with resolved types, nullable flags, and output_schema. Include tests for column resolution, ambiguity detection, type coercion, aggregate-in-WHERE rejection, GROUP BY validation, wildcard expansion, view expansion, correlated subquery scoping.

Target files: sqld/src/sql/binder.rs, sqld/src/sql/bound_ast.rs, sqld/tests/test_binder.rs
