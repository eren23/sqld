# Task: SQL parser with Pratt expressions and full AST

Implement Pratt parser (top-down operator precedence) for expressions with 14 precedence levels (OR lowest through function call highest). Implement recursive descent for all statements: SELECT (DISTINCT, columns, FROM with joins/subqueries/aliases, WHERE, GROUP BY, HAVING, ORDER BY with ASC/DESC/NULLS FIRST|LAST, LIMIT, OFFSET, set operations UNION/INTERSECT/EXCEPT), INSERT (column list, VALUES single/multi-row, INSERT...SELECT), UPDATE (SET, WHERE, RETURNING), DELETE (WHERE, RETURNING), CREATE TABLE (all column constraints: NOT NULL/DEFAULT/PRIMARY KEY/UNIQUE/CHECK/REFERENCES with ON DELETE/UPDATE actions, table constraints), DROP TABLE (IF EXISTS, CASCADE), ALTER TABLE (ADD/DROP/RENAME COLUMN, ADD/DROP CONSTRAINT), CREATE/DROP INDEX (UNIQUE, USING HASH, ASC/DESC), CREATE/DROP VIEW, BEGIN/COMMIT/ROLLBACK/SAVEPOINT, EXPLAIN [ANALYZE], SHOW TABLES/COLUMNS, ANALYZE, VACUUM, COPY. Implement full Expr enum and Statement enum as specified in the AST. Implement error recovery: skip to next statement boundary on error, collect multiple errors with source location and expected tokens, common mistake detection. Include tests for all statement types, operator precedence, subqueries, CASE expressions, set operation precedence, error recovery, and diagnostics quality.

Target files: sqld/src/sql/ast.rs, sqld/src/sql/parser.rs, sqld/tests/test_parser_select.rs, sqld/tests/test_parser_dml.rs, sqld/tests/test_parser_ddl.rs, sqld/tests/test_parser_expressions.rs, sqld/tests/test_parser_errors.rs
